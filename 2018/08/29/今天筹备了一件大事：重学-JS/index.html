最近在阮大神的博客上看到一篇文章，讲的是关于如何自学计算机技术，原文出自 Teach Yourself Computer Science。看完以后我明白自己的缺陷在哪里，基础不够牢固是我最大的问题。

经过多年的摸索，学以致用是学习最好的方式，这个已经深入我心，所以在对基础学习的过程中，我准备边学边用。

在 github 上建了一个新的项目，名字叫做：javascript-learning-guide，启发来源于一个 Node 学习笔记，希望我的学习过程也能影响一部分人，特别是自学前端的同学，毕竟每个人都是从小菜鸟变成老鸟的。

项目地址：https://github.com/kurryluo/javascript-learning-guide.git

这个 github 项目主要讲解的一个个的知识点，每一个阶段结束后，我会把可以运行的实战项目公布出来，供大家参考和学习。

第一个实战项目是课题组网站，敬请期待。

欢迎大家关注微信公众号：可视化技术（ visteacher ）

不仅有前端和可视化，还有算法、源码分析、书籍相送

个人网站：http://blog.kurryluo.com
