前言（着急的可以跳过）

我本科不是计算机专业的，研究生也不是。但这一路走来，我逐渐发现计算机对各行各业都具有深远的影响，特别是在研究了深度学习之后，我对计算机的痴迷程度从原来只想把计算机做为展示数据展示工具，到如今我想深入地了解计算机是如何帮助人们生产、生活的。

网上的一些优秀的开发人员一开始也不是计算机专业的，比如，前端开发的阮一峰老师。大学的教育让很多计算机专业的同学对计算机产生了恐惧，反而是从没深入了解过的非专业的学生发现计算机的强大以后主动学习，自主学习这股动力像一阵风，开发技术就像草，见风就长。所以，非计算机的同学不用害怕，喜欢就上，人就得干点自己喜欢又带有挑战性的事情。

接下来，让我们搭建好代码住的地方。

环境

IntelliJ Idea

IDEA 主要用于 JAVA 的开发，当初选择这个集成环境，是因为我们公司的后端使用 JAVA 进行开发，所以大家都安装这个编译器。IDEA 官网的下载地址。安装对应版本后，需要注册。有条件的可以选择购买注册码，当前价格是第一年折合人民币 3500元 。不过我们还有另外的选择，lanyu 提供的注册码满足了大部分人的需求。

安装步骤跟普通软件的无异，这里给大家提供一个由极客学院提供的比较详细的 IDEA 使用教程，里面涉及了很多使用 IDEA 的技巧。不过，一个软件最好的参考资料是官方文档，我现在看完基础的使用教程后，也逼着自己看官方文档，你会发现 IDEA 比其他的开发工具更加智能，更加人性化。一些快捷键等到需要用的时候再学习，这样会高效一些。

Node.js

前端的三把剑是 JavaScript(JS)、Css、Html，Node.js 是在服务端运行的 JS 。Node 的下载地址选择适合自己电脑的版本安装，配置好环境变量。然后在 IDEA 中在 Intellij IDEA 中安装 node plugins。打开“文件”菜单，选择“设置”，选择 “Plugins” ，输入 “nodeJS” ,再点击安装即可。

安装 Node.js 的同时，npm 作为安装包管理工具也会一起被安装，在 IDEA 中新建一个项目，在新建的项目中使用命令npm -v，就可以检测是否安装成功。如下代码：

E:\work\newproject>npm -v

Git

Git 被用来帮助人们管理代码的版本，其使用教程可以单独写一本小册子，这里推荐「stormzhang」的 Git 速成。我在学习 React 的时候使用的是阮一峰老师的全栈工程师培训教程，包含一个例子集合，浅显易懂，这是地址，可以试着在 Idea 中直接 git clone 代码。

总结

环境的搭配是每个程序员的必修课，我刚开始也反感这些繁杂的环境，甚至会想，要是有家公司来个大礼包，省去这些步骤该多好。但是，当搭建成功的时候，那种成就感就是程序员热爱这行的原因吧。

下一篇文章，或者说以后的文章，我都会针对很小的一个点进行描述。这篇我自己感觉太着急，太赶了。不过，慢慢会好的。
